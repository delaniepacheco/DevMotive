# DevMotive
Project 
#DevMotive

#Overview 
DevMotive is a beginner-friendly Python web app designed to help new programmers stay motivated while leanring to code.
#Purpose
Learning to code can be overwhelming, espiecially for beginners. DevMotive aims to reduce frustration and encourage consitency by offering positive reinforcement and small manageable learning tasks and tips and tricks. 

#Features
-Generate motivational messages
-Generate Python tips
-Generate debugging advice
-Generate coding challeneges
-Save and favorite messages
-Simple and easy to use interface

#This appliction is meant for beginner programmers that need encouragement 

#Technologies used
-Python
-PyWebIO
